Website yang saya buat adalah sebuah sistem arsip cerita horor digital, di mana pengguna dapat membaca, menulis, dan mengelola berbagai cerita mistis yang berasal dari pengalaman pribadi maupun urban legend. Admin atau pengguna dapat melakukan proses seperti menambahkan cerita baru, melihat detail cerita, mengedit, menyetujui, atau menghapus cerita. Sistem ini seperti alur moderasi konten seperti pada platform berbagi cerita, tapi dengan nuansa horor.
